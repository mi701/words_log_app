# log. - Fact-Based Journaling (PWA)

「評価しない」「自己内省を促さない」
ただ、今の状態を置いていく。

## 概要
log. は、感情的な吐露や自己分析を目的としない、極めて客観的な記録（ジャーナリング）アプリケーションです。身体感覚や周囲の状況など、「事実」を表す言葉を配置していくことで、評価や改善というプレッシャーから離れた「今この時」を静かに残します。

## 主な特徴

- **思想に基づいた語彙セット**: 感情や評価を排し、身体（体・頭）、環境（気配・光・音）、感覚（質感・距離・時間・重力）といった描写に特化した語彙を搭載。
- **選択＝記録の操作フロー**: 複雑な入力や「保存」という決断すら最小化。直感的に言葉を選び、配置するだけで記録が完成します。
- **AI による語彙の拡充**: Gemini API を連携させることで、選んだカテゴリに基づいた新しい言葉を AI が提案してくれます。（API キーがなくても内蔵語彙だけで利用可能です）
- **ローカル保存**: データはブラウザの IndexedDB にのみ保存され、外部サーバーに送信されることはありません（Gemini API 利用時の単語生成リクエストを除く）。

## 使い方
1. **記録**: カテゴリを選び、今の状態に近い言葉をタップしてキャンバスに配置します。配置が終わったら完了ボタンで保存します。
2. **観測**: 過去の記録を客観的に眺めることができます。何かの傾向を探るのではなく、ただ「そういう事実があった」ことを確認する場所です。
3. **設定**: Gemini API キーを設定することで、AI による語彙生成が可能になります。
